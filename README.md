#  Эмодзи-викторина «Угадай по смайликам»

Десктопная викторина на C# (WinForms + SQLite): на экране показывается набор эмодзи,
игрок угадывает по ним название фильма, мультфильма или сериала. Данные для викторины
подготовлены из настоящих открытых датасетов (Kaggle), а не написаны руками.

# Что умеет приложение

- Главное меню с выбором: играть или зайти в режим администратора.
- Режим игры: случайный вопрос (эмодзи), четыре варианта ответа, подсчёт счёта, переход к следующему вопросу.
- Фильтр по категориям (Фильм / Мультфильм / Сериал).
- Режим администратора: добавление новой записи (эмодзи, ответ, категория) — она сразу участвует в игре.
- Все вопросы хранятся в базе данных SQLite, а не в коде. Записи администратора сохраняются между запусками.
- Вопросы и варианты ответа каждый раз выбираются случайно.

# Стек технологий

| Часть | Технология |
|---|---|
| Подготовка данных (этап 1) | Python 3, pandas |
| Приложение (этап 2) | C#, .NET 10, WinForms |
| База данных | SQLite через Entity Framework Core |

# Откуда данные

1. Английские названия фильмов и подобранные к ним эмодзи взяты из датасета
   [Movies to Emojis](https://www.kaggle.com/datasets/tarundalal/movies-to-emojis-dataset) на Kaggle.
2. Русские названия фильмов взяты из [официальных датасетов IMDb](https://datasets.imdbws.com/).
3. Оба источника связаны по идентификатору фильма IMDb (`tconst`, например `tt0110357`).
4. На выходе — чистая таблица `movies_ru_emoji.tsv` из четырёх колонок: идентификатор,
   русское название, эмодзи, категория. Она и загружается в базу приложения при первом запуске.

Скрипт подготовки данных: `prepare_data.py`.

#  Как запустить

# Требования

- [.NET 10 SDK](https://dotnet.microsoft.com/download)
- Windows (WinForms работает только под Windows)
- Python 3 с библиотекой pandas — нужен только если хотите пересобрать `movies_ru_emoji.tsv` заново;
  готовая таблица уже лежит в репозитории, так что для простого запуска игры Python не обязателен.

# Запуск приложения

1. Клонируйте репозиторий:

   ```bash
   git clone https://github.com/voskoboinik/emoji-quiz.git
   cd emoji-quiz/EmojiQuiz/EmojiQuiz
   ```

2. Запустите проект:

   ```bash
   dotnet run
   ```

   При первом запуске автоматически создастся файл базы `quiz.db` и заполнится
   данными из `movies_ru_emoji.tsv`. При последующих запусках база не пересоздаётся —
   данные пересобираются из tsv-файла заново, только если удалить `quiz.db` вручную.

3. В главном меню выберите **«Играть»**, чтобы начать викторину, или **«Администратор»**,
   чтобы добавить свой вопрос.

# Пересборка данных (по желанию)

Если хотите пересобрать `movies_ru_emoji.tsv` из исходных источников:

```bash
pip install pandas
python prepare_data.py
```
